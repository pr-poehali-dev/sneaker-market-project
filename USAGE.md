# SneakerLab - Руководство по использованию

## Основные возможности сайта

### 1. Главная страница
- **Hero секция** с интерактивным слайдером сравнения оригинала и реплики
- **Блок преимуществ** - гарантия, доставка, обмен
- **Как это работает** - 4 шага от выбора до получения

### 2. Каталог товаров
- Фильтрация по категориям: Все / Оригинал / Реплика
- Индикатор сходства для реплик
- Добавление в корзину через выбор размера
- Hover-эффекты на карточках

### 3. Корзина
- Открывается через кнопку в навигации
- Изменение количества товаров
- Удаление товаров
- Подсчет общей суммы
- Переход к оформлению

### 4. Оформление заказа (3 шага)
**Шаг 1:** Контактная информация
- Имя и фамилия
- Телефон
- Email

**Шаг 2:** Доставка
- Адрес доставки (город, адрес)
- Выбор способа доставки:
  - Курьерская доставка (2-3 дня, бесплатно)
  - Самовывоз из магазина (сегодня, бесплатно)
  - Экспресс доставка (на следующий день, 1500₽)

**Шаг 3:** Оплата
- Банковская карта
- Наличными при получении
- Система быстрых платежей (СБП)

### 5. Отслеживание заказа
- Поиск по номеру заказа или трек-номеру
- Визуальная временная шкала статусов
- Информация о товарах в заказе
- Адрес доставки
- Ожидаемая дата доставки

### 6. Тест-драйв реплики
- Интерактивный калькулятор скидки
- Выбор срока тест-драйва (1-2 дня)
- Расчет скидки на обмен на оригинал (15-20%)

### 7. Программа лояльности
Три уровня:
- **Bronze** (0₽) - скидка 5%, бесплатная доставка
- **Silver** (от 30,000₽) - скидка 10%, приоритетная обработка
- **Gold** (от 70,000₽) - скидка 15%, персональный менеджер

### 8. Навигация
- Десктоп: полное меню в шапке
- Мобильная версия: бургер-меню
- Плавная прокрутка по якорям
- Кнопка "Наверх" при прокрутке вниз

## Технические особенности

### Адаптивность
- Полная поддержка мобильных устройств
- Адаптивная типографика (от 4xl до 7xl)
- Оптимизированные изображения для разных экранов

### Анимации
- Fade-in эффекты при загрузке
- Scale-in для главного изображения
- Glow-pulse для CTA кнопок
- Hover-эффекты на карточках и кнопках
- Плавные переходы между состояниями

### UX улучшения
- Счетчик товаров в корзине
- Toast-уведомления при действиях
- Валидация форм
- Loading состояния
- Плавная прокрутка

## Горячие клавиши
- **ESC** - закрыть модальные окна
- **Прокрутка вниз >300px** - появление кнопки "Наверх"

- Метатеги для поисковых систем

- Семантическая разметка HTML
- Оптимизированные заголовки
